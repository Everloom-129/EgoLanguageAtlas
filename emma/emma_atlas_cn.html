第一人称示例媒体

示例视频 · 项目主页
EMMA, Zhu et al., Georgia Tech, RA-L 2026

EMMA 用 Meta Project Aria 眼镜采集第一人称人类移动操作演示，记录宽视场 RGB 以及 3D 手部与头部姿态。它将人类头部轨迹重定向为运动学可行的移动底盘指令，并在相机相对坐标系中对齐操作，然后在这些人类全身运动数据与静态（非移动）机器人遥操作数据上协同训练单一的多头 Transformer 策略，从而免去移动遥操作。最终得到面向双臂差速移动机器人的移动操作策略。EMMA 的监督来自运动与轨迹数据而非语言，其代码标注为即将发布。

04为何为描述性条目

Verified via arXiv 2509.04443 and ego-moma.github.io (BibTeX zhu2026emma, IEEE RA-L 2026). Code and data are listed as coming soon; no license is stated.

EMMA 是一种移动操作协同训练方法，而非标注来源。其人类数据携带的是运动与轨迹信号（手部、头部与底盘姿态）而非语言，所预测的子任务标签也是类别标签而非自由文本。代码与数据标注为即将发布，因此本图谱以描述性方式记录 EMMA，而非对下载的字符串进行聚类。
