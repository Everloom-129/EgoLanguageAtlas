01Data-usage path

Example egocentric media
VIP is pretrained on unlabeled Ego4D egocentric video; the project page shows example footage and learned reward curves.
example frame · example video · project page
VIP, Ma et al., ICLR 2023

VIP consumes Ego4D egocentric video without using any action or language labels. It casts representation learning as offline goal-conditioned reinforcement learning and derives a self-supervised dual value-function objective (an implicit time-contrastive loss) trained over video frames. It releases pretrained visual-encoder weights that yield both a representation and a dense, smooth reward (the embedding distance to a goal image), not a new language-annotation corpus.

04Why this is a descriptive entry

Verified via arXiv 2210.00030 and github.com/facebookresearch/vip (CC BY-NC 4.0); project page sites.google.com/view/vip-rl.

VIP is a reward and representation-learning method, not an annotation source. It trains on unlabeled Ego4D frames (it does not consume narrations) and releases model weights under CC BY-NC 4.0. There are no VIP-original language strings to download or cluster, so this atlas documents it descriptively.
