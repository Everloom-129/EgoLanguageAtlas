01Data-usage path

Example egocentric media
example frame
EgoDex: 829 hours of Apple Vision Pro egocentric manipulation with 3D hand pose.
example frame · example video · project page
EgoDex, Hoque et al., ICLR 2026 (CC BY-NC-ND, internal analysis).

EgoDex captures egocentric manipulation with Apple Vision Pro, pairing video and native 3D hand pose with a per-episode task instruction generated by an LLM and stored as an HDF5 attribute. This atlas reads the instruction attributes from the test split (no video downloaded) and clusters the active instruction per episode. Reversible tasks store both directions (for example add a lid versus remove a lid); the direction actually performed is selected per episode.
