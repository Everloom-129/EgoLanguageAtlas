LAPA, Ye et al., ICLR 2025

LAPA trains an action-quantization model that infers discrete latent actions between consecutive frames of actionless video, including human manipulation video such as Something-Something V2. It then pretrains a latent vision-language-action model to predict those latent actions from an observation and a task description, and fine-tunes on a small set of real robot trajectories to map latent actions to executable ones. LAPA releases the latent-action model and a 7B VLA under MIT; the task descriptions it conditions on are consumed from existing datasets, so it publishes no new language corpus.

04Why this is a descriptive entry

Verified via arXiv 2410.11758, OpenReview (ICLR 2025), github.com/LatentActionPretraining/LAPA (MIT), and huggingface.co/latent-action-pretraining. Task instructions are consumed from source datasets, not newly authored.

LAPA is a representation- and policy-pretraining method, not an annotation source. The latent actions are learned without action labels, and the task instructions it uses as conditioning are reused from Open X-Embodiment, BridgeV2, and Something-Something V2 under their own licenses rather than authored anew. It releases code and weights under MIT, so this atlas documents LAPA descriptively rather than clustering downloaded strings.
