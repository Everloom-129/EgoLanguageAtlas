01数据使用路径

第一人称示例媒体
VRB 从 Ego4D 与 EPIC-KITCHENS 第一人称视频中提取接触点与接触后腕部轨迹；项目主页与演示视频展示了可供性预测。
示例帧 · 示例视频 · 项目主页
VRB, Bahl et al., CVPR 2023

VRB 使用 Ego4D 与 EPIC-KITCHENS 的第一人称人类视频，检测手物接触帧并跟踪腕部，提取接触点与接触后轨迹。它将这些信息映射到去除人类的帧（将人类修复抹去），训练一个直接从图像预测接触热图与轨迹的视觉可供性模型。其输出是视觉可供性而非语言，因此不产生可聚类内容。

Ego4D + EPIC-KITCHENS 视频
→
接触检测 + 腕部跟踪
→
去除人类的帧映射
→
视觉可供性模型（接触 + 轨迹）
→
机器人学习（模仿 / 探索 / 目标条件 / 动作参数化）

04为何为描述性条目

Verified via arXiv 2304.08488, vision-robotics-bridge.github.io, and github.com/shikharbahl/vrb (MIT).

VRB 是一种可供性学习方法，而非标注来源。它消费已有的第一人称视频，并以 MIT 许可发布代码与一个输出接触热图和腕部轨迹（而非文本）的可供性模型。没有 VRB 原创的语言字符串可供下载或聚类。
